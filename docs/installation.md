# Installation

## Stable release

To install py-cdp-rx-flow-bot, run this command in your terminal:

```sh
uv add py-cdp-reactive-flow-bot
```

Or if you prefer to use `pip`:

```sh
pip install py-cdp-reactive-flow-bot
```

## From source

The source files for py-cdp-rx-flow-bot can be downloaded from the [Github repo](https://github.com/yishenggudou/py_cdp_reactive_flow_bot).

You can either clone the public repository:

```sh
git clone git://github.com/yishenggudou/py_cdp_reactive_flow_bot
```

Or download the [tarball](https://github.com/yishenggudou/py_cdp_reactive_flow_bot/tarball/master):

```sh
curl -OJL https://github.com/yishenggudou/py_cdp_reactive_flow_bot/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd py_cdp_reactive_flow_bot
uv pip install .
```
